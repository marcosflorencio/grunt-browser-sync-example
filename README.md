# grunt-browser-sync-example
Exemplo de browser-sync utlizando uma task no grunt

## Instalando
```
$ npm install
$ npm install -g grunt-cli (caso nunca tenha usado o grunt)
```

## Colocando para funcionar
```
$ grunt browser-sync
Abra http://localhost:3000 :)
```
